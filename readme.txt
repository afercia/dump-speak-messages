== Description ==

Dump Speak Messages is a small plugin for debugging purposes. It dumps the wp.a11y.speak() messages to the browser console.

== Installation ==

1. Upload the `dump-speak-messages` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
